# UNO Game in Java with Custom GUI

## Description
This repository contains a fully functional **UNO card game** implemented in **Java**, featuring a **customized graphical user interface (GUI)** built on top of the Swing library. The project demonstrates **object-oriented programming principles** such as inheritance, polymorphism, and encapsulation through a dedicated game library of reusable components.  

The game allows multiple players, handles the full set of UNO rules including special cards (Reverse, Skip, Draw Two, Wild, Wild Draw Four), and provides interactive GUI elements for player actions, game history, and opponent displays.  

Key highlights of the project:  
- **Custom GUI Layer:** All Swing components (JFrame, JButton, JPanel, JTable, etc.) are extended and customized for a consistent look and feel, making the GUI intuitive and visually appealing.  
- **Game Logic Integration:** The UNO rules and game flow are fully implemented, allowing for smooth player interactions and real-time updates of hands, opponents, and the game table.  
- **Object-Oriented Design:** Classes are modular and well-organized, separating GUI components, game logic, and player interactions.  

This project serves as both a practical learning example for Java GUI programming and an engaging UNO game experience.
